Project Overview:-
Mood Muse is an AI-powered emotional wellness web application that helps users understand and reflect on their daily emotions. It provides supportive and
empathetic insights based on the user's selected mood.

Objective:-
- Help users identify their emotional state.  
- Promote emotional awareness.  
- Encourage healthy responses like mindfulness and reflection.  

Features:-
- Mood selection system.  
- AI-powered chatbot for emotional support.  
- Personalized mood-based insights.  
- Tracks mood history for better understanding.  
- Clean and responsive UI.  

Technologies Used:-
- HTML  
- CSS  
- JavaScript
- Lovable AI Platform

Methodology:-
- User-centered design approach.  
- Mood-based logic implementation.  
- Dynamic content rendering.  
- AI integration for chatbot support.  

Output:-
- Users select mood → get insights instantly.  
- Dynamic suggestions based on emotions.  
- AI chatbot provides supportive responses.  

Result:-
- Fully functional emotional wellness app.  
- Real-time personalized insights.  
- Improved user engagement through AI.  

Conclusion:-
Mood Muse shows how technology can support emotional well-being in a simple and effective way. It provides a clean, easy-to-use platform for emotional reflection.
